1. Qual é a declaração de variável correta em JavaScript?
a) let minhaVariavel = 10;
b) variavel minhaVariavel = 10;
c) const minhaVariavel == 10;
d) var = minhaVariavel 10;

✅ Resposta: a) let minhaVariavel = 10;

2. Qual destas estruturas faz um laço de repetição?
a) if
b) switch
c) while
d) function

✅ Resposta: c) while

3. Como se escreve um comentário de uma linha em JavaScript?
a) <!-- Este é um comentário -->
b) /* Este é um comentário */
c) // Este é um comentário
d) ** Este é um comentário **

✅ Resposta: c) // Este é um comentário

4. Qual é o operador de igualdade estrita em JavaScript?
a) =
b) ==
c) ===
d) !==

✅ Resposta: c) ===

5. Como se declara uma função em JavaScript?
a) function minhaFuncao() { }
b) def minhaFuncao() { }
c) fun minhaFuncao() { }
d) define minhaFuncao() { }

✅ Resposta: a) function minhaFuncao() { }
